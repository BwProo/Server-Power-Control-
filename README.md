# 🖥️ Server Power Control

A [Pelican Panel](https://pelican.dev) plugin by **BwProo** that lets admins start, restart, stop, or kill multiple game servers at once — directly from the admin panel.

---

## ✨ Features

- ✅ Select individual servers or all at once
- ▶️ **Start** selected servers
- ↺ **Restart** selected servers  
- ■ **Stop** selected servers
- ✕ **Kill** selected servers (force stop with confirmation)
- 🔔 Toast notifications (top-right) with icons after every action
- 🔗 Shows server name, owner, node and UUID

---

## 📦 Installation

1. Download the latest release ZIP
2. Upload and install via **Admin → Plugins → Install Plugin**
3. Enable the plugin
4. Navigate to **Admin → Server Power Control**

---
